# Guojiz.github.io

Personal research homepage for 郭己正 / Guojiz, focused on non-sovereign superintelligence, mirror deterrence, and human fallback continuity.

The site is a dependency-free bilingual experience designed to deploy directly through GitHub Pages. A large argument layer and a fixed translucent transcript move in parallel over one continuous Canvas particle timeline. The research document below uses scroll-revealed cards and includes a bilingual glossary of the theory's specialized terms.

Phase-one research formally begins in July 2026. The site invites narrow expert critique, literature guidance, conceptual review, and sustained scholarly collaboration across AI safety, AI governance, deterrence theory, political theory, human factors, and research ethics.

Key files:

- `index.html`: semantic bilingual content
- `styles.css`: responsive visual system
- `particles.js`: deterministic particle narrative
- `site.js`: language and scroll state
- `PARTICLE_STORYBOARD.md`: narrative and implementation blueprint
- `brand-spec.md`: identity, color, type, and motion rules

Live URL: <https://guojiz.github.io/>
